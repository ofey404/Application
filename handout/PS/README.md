PS
===

本目录存放个人陈述。

包括个人陈述的草稿、片段和成品。

会使用 `.zh` `.en` 后缀来标明语言，相同基础名的文件为相关联的内容。

尽量使用英文撰写，在需要迅速记录思路的时候会使用中文。

我的英文水平一般，并且在仓库中写作时没有拼写检查。勘误和润色会放到最后一步去做。
